# proyecto-final-de-inventario
este proyecto nacio de una idea por comprar papas de la tienda que esta al frente de la universidad y les preguntamos a las señoras como sabian si no es faltaban productos y que les llega, a lo que nos dijeron que no sabian que solo recordaban algunas cantidades y se hacian lios en momentos ya que no se acordaban que entraba y que vendian y asi nacio la idea inicial para que cualquier comerciante mayorista pequeño y grande sepa que es lo que tiene 
# QUE ES ?
lo que este codigo con grafica quiere lograr es que se pueda utilizar lo aprendido en clases con el ingeniero ruben y el auxiliar de ayudantias ya que se maneja lenguajes que manejamos en clases y usando python y vs code ademas que github para compartir el proyecto con los diferentes integrantes de grupo
# FUNCIONAMIENTO
este codigo hace las siguientes cosas
## RECOPILA DATOS 
toma lo que los usuarios escriben y lo guarda en (codigo,nombre,cantidad,precio y buscador)
## VALIDACION DE INFORMACION 
tiene filtros de seguridad (if y elif) para comprobar que no dejes campos obligatorios,ademas que sean unidades enteras 
# INTERFAZ Y ACTUALIZACION VISUAL 
esto se divide eun 2 campos
## LIMPIEZA AUTOMATICA
despues de cada accion exitosa (agregada o vendida) el programa vacia las cajas de texto que esten listas para la siguiente orden 
## REFRESCO EN TIEMPO REAL
despues de cada o cualquier cambio se borran los datos viejos de la tabla visual de la derecha y la vuelve a dibujar con la informacion actualizada
